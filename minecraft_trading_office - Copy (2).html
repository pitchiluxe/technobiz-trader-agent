Usage Guide
1. Add Core Agents
Click "+ Agent" and add these 3 agents:
• Trend-Master (Costume: Classic)
• Analyse-Master (Costume: Worker)
• Trader-Master (Costume: Dark)
2. Select Trading Pair
Enter a pair (EURUSD, BTCUSD, etc.)
3. Start Trading Cycle
Click "START CYCLE" to begin. All 3 agents must be seated at their desks.
4. Watch the Workflow
• Agents light up at their desks
• Monitor glows while working
• Speech bubbles show live status
• Agents return to idle at desk when done
💡 Tip: Use Zoom (+/-) to adjust view size
